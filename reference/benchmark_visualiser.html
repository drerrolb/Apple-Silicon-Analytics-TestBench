Why Apple Silicon Wins This Workload
CPU · Python
Sequential scoring
pandas processes each row serially through the Python interpreter. The GIL prevents true parallelism. A 1,000-record batch is fundamentally a loop.
Memory · Python
Copy-heavy pipeline
DataFrames are duplicated on merge. 10M rows at ~320 bytes each consumes 3+ GB with working copies. GC pressure spikes P99 latency.
GPU · Metal
Parallel scoring
The Metal compute shader dispatches 1,000 threads simultaneously — one per transaction. Every record in the batch is scored in a single GPU pass.
Memory · Apple Silicon
Zero-copy unified
CPU-written buffers are directly readable by the GPU. No PCIe transfer. No VRAM copy. The result buffer lives at the same physical address both sides read.
